업로드 방법

1. 이 폴더 안의 파일들을 모두 웹사이트 루트 경로에 업로드합니다.
   예: /public_html/, /www/, /htdocs/

2. 업로드 후 아래 URL이 열리는지 확인합니다.
   https://itaewonclub.kr/robots.txt
   https://itaewonclub.kr/sitemap.xml
   https://itaewonclub.kr/rss.xml
   https://itaewonclub.kr/feed.xml
   https://itaewonclub.kr/llms.txt
   https://itaewonclub.kr/llms-full.txt
   https://itaewonclub.kr/seo-text.txt

3. Google Search Console에서 sitemap.xml을 제출합니다.
4. 네이버 서치어드바이저에서 sitemap.xml을 제출하고, 필요 시 rss.xml도 제출합니다.
5. head-feed-tags.txt는 선택 사항입니다. RSS/Atom 발견성을 높이고 싶을 때 각 HTML의 <head>에 복사해 넣습니다.

주의: 이 파일들은 크롤링과 정보 발견성을 개선하는 보조 파일입니다. 검색 상위노출을 보장하지는 않습니다.
